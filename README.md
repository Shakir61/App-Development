# App-Development
This is my app development learning journey in which i will share all the topics and projects that i will work on.

Building my first app:
First of all, confirm your dart environment:
run "dart --version" command in command prompt.

Then create a new directory for your project:
run "mkdir dartpedia"
then "cd dartpedia"

Run the following command:
"dart create cli"
The dart create command generates a basic Dart project named "cli" (for Command Line Interface). It sets up the essential files and directories you need.

In the terminal, navigate into your new project directory:
"cd cli"
Run the default application:
"dart run"
